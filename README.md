# 💰 Assistente Financeiro Pessoal

Um sistema de gerenciamento financeiro desenvolvido em Python para o acompanhamento de despesas diárias, focando na organização de dados via matrizes e automação de relatórios.

## 🎯 Objetivo do Projeto
Automatizar o registro de gastos e proporcionar uma visão clara do orçamento doméstico através de um terminal interativo. O projeto visa aplicar conceitos fundamentais de programação como funções, estruturas de repetição e manipulação de listas.

## 🚀 Funcionalidades Principais

- **[1] Cadastro de Transações:** Registro detalhado com descrição, valor (suporta decimais) e categoria.
- **[2] Extrato Detalhado:** Exibição organizada de todas as movimentações financeiras.
- **[3] Relatório de Impacto:** Análise de gastos por categoria para maior controle financeiro.

## 🛠️ Tecnologias Utilizadas
- **Python 3:** Linguagem base com foco em modularização (uso de funções e `if __name__ == "__main__":`).
- **Estruturas de Dados:** Matrizes para armazenamento persistente durante a execução.
- **Git/GitHub:** Controle de versão e organização de portfólio.
